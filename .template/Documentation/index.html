CSS Files and Structure

I'm using seven CSS files in this theme. The First CSS is bootstrap.min.css. This is the base CSS for Bootstrap. I have used bootstrap 3.2.0. If you need you can use minified version instead of deveelopment version.

The second CSS "style.css its style using for stylings for the page. The file is separated into sections using:

/* Table OF Contents
==========================
1.Banner area
2.Advantage
3.Video
4.Service
5.Offer
6.Price
7.Client
8.Footer
=========================== */

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

10. respond.min.js include script for ie supporting file
